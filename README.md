## Pré-requisitos

Para compilar e executar o projeto, você precisará do compilador g++ e da biblioteca SFML instalados em seu sistema. Certifique-se de ter os seguintes arquivos:

- `fila.cpp`
- `musica.cpp`
- `lista.cpp`
- `algoritmos.cpp`
- `main.cpp`
